## 付费类型分类

### 1. Azure Microsoft 系列 (约20个)
需要 Azure 账号和相应的 API 服务，包括：
- Azure OpenAI API
- Azure Storage
- Azure App Insights
- Azure Monitor
- Azure Active Directory (Entra ID)
- Azure Resource Graph

### 2. AI 图像/视频生成 (约20个)
需要付费的 AI API：
- OpenAI DALL-E
- 阿里云通义千问
- Google Gemini
- Stability AI
- Midjourney (可能)

### 3. 社交媒体 API (约5个)
- Twitter/X API
- 微信 API

### 4. 数据库服务 (约2个)
- Neon (PostgreSQL)
- Supabase

### 5. 其他专业服务 (约3个)
- Firecrawl (网页爬虫)
- Remotion (视频渲染)
- SeDance2 (自动化)

## 总结

在 Top 100 Skills 中，大约 **50% 需要付费**。

主要付费类型：
1. **Azure 云服务** - 微软生态
2. **AI API** - 图像/视频/文本生成
3. **社交媒体 API** - Twitter、微信
4. **数据库服务** - Neon、Supabase
5. **专业工具 API** - 爬虫、视频、自动化
