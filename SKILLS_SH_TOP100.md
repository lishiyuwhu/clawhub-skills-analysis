## 结论

Skills.sh 的 Top 100 呈现以下特点：

1. **微软 Azure 主导** - 20+ Azure 相关技能占据高位
2. **Vercel 生态丰富** - 前端最佳实践系列
3. **AI 能力是付费热点** - 图像/视频生成需要 API
4. **企业导向明显** - 云服务、数据库、DevOps

对比 ClawHub：
- Skills.sh 更偏向**企业级云服务**
- ClawHub 更偏向**个人开发者工具**
